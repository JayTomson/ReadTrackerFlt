
# ReadTracker

Минималистичный локальный трекер для чтения ранобэ (Light Novels), веб-новелл (Web Novels) и синглов на Flutter. Полностью автономное решение без громоздких внешних баз данных — весь ключевой функционал оптимизиован для работы на одном устройстве.

## Функционал

* **Гибридный формат (LN+WN):** одновременный учет томов ранобэ и глав веб-новеллы внутри одной карточки.
* **Гибкие метрики:** подсчет прогресса в словах, томах (с возможностью детальной разбивки) или главах.
* **Кастомные закладки:** поле актуальной закладки выносится прямо на главный экран библиотеки.
* **Точка старта адаптации:** фиксация тома/главы, с которой начато чтение, чтобы точно разделять прочитанное в приложении и за его пределами.
* **Система рейтингов:** переключаемая шкала оценки тайтлов (на 5 или 10 звезд).
* **Режимы отображения:** компактный текстовый список с цветовыми метками или карточки с обложками (поддерживаются локальные файлы и сетевые URL).
* **Экспорт и импорт:** резервное копирование и восстановление всей базы данных в формате JSON через встроенный файловый менеджер.
* **Модуль аналитики:** автоматический подсчет суммарного объема слов, томов и прогресса по статусам («Читаю», «В планах», «Завершено», «На паузе», «Брошено»).

## Технические детали

* **Архитектура состояния:** управление через реактивный `AppState` (`ChangeNotifier`) с автоматической сериализацией и сохранением изменений в `SharedPreferences`.
* **Оптимизация рендеринга:** навигационные экраны изолированы через `IndexedStack` для экономии ресурсов процессора и сохранения состояния скролла при переключении вкладок.
* **Дизайн-токены:** жестко зафиксированные константы верхнего уровня (`kAccent`, `kR`, `kP`) гарантируют визуальную целостность UI без раздувания кода стилей.
* **Кастомизация интерфейса:**
  * Три режима отображения интерфейса (светлый, темно-серый и True Black для OLED-экранов).
  * Возможность полного отключения анимаций переходов для моментального отклика на слабых устройствах.
  * Принудительный нативный скроллинг `BouncingScrollPhysics` на всех платформах через кастомный `ScrollBehavior`.
